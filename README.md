### Beginner Functions
* data type : var, let, const

```
let letName = "홍길동";
letName = "김철수";
console.log(letName);

const constName = "홍길동";
constName = "김철수"; // 재할당 불가
```

* variable type : primative(String, Number,Boolean, Null etc), Object(array, map etc)
* conditions : boolean, if, switch
* loops : while, forEach
* function : basic, hoisting
